# Insurance Cost Analysis Dashboard

## Project Overview
This project analyzes a public medical insurance dataset from Kaggle to explore factors affecting insurance charges. The analysis focuses on smoking status, BMI, age, region, and customer risk categories using Tableau dashboards.

## Tools Used
- Tableau
- Excel
- SQL
- Kaggle Dataset

## Analysis Focus
- Smoking impact on charges
- BMI vs insurance cost
- Age-group analysis
- Regional comparison
- Risk segmentation

## Key Insights
- Smokers showed significantly higher average insurance charges.
- Higher BMI levels were associated with increased medical costs.
- Insurance charges increased with age.
- High-risk customers contributed disproportionately to total costs.
